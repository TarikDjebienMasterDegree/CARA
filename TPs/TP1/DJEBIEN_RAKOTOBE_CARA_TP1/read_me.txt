Ce fichier est un fichier readme pour le tp1 de CARA,
de la formation MIAGE en alternance, 2ème année.


Le tp a été réalisé en binôme dont les auteurs sont :
- TARIK Djebien
- RAKOTOBE Eric


Tout ce qui a été proposé dans le TP a été fait.


Dans cette archive fournie, on retrouve 3 fichiers :
	- le fichier read_me.txt ci-ouvert
	- un 'tutoriel' expliquant en détail comment le TP a été réalisé.
	- le code source du TP.


Dans le code source du tp, on retrouve :
	- Les tests unitaires rédigés pour tester les différentes méthodes 
	dans : Axis2WebServices\src\com\client\test

	- les différents webservices développés, ainsi que les stubs et handlers
	autogénérés par Eclipse/Axis2
	dans : Axis2WebServices\src\com\webservices
